# ModernResume Studio

A powerful, client-side resume builder designed to create professional resumes with ease. It features a schema-driven form builder, real-time live preview, and high-quality PDF export, all wrapped in a modern, responsive interface with glassmorphism effects.

## Features

- **Schema-Driven Forms**: Automatically generates form inputs based on a JSON schema configuration defined in JavaScript.
- **Live Preview**: See changes instantly as you type with a split-screen view.
- **Multiple Templates**: Switch between 'Minimal' and 'Card' layouts on the fly without losing data.
- **PDF Export**: Download your resume as a high-quality PDF (powered by `html2canvas` & `jspdf`).
- **Theming**: Includes a toggleable Dark/Light mode and multiple preview background styles (Plain, Grid, Slate).
- **Responsive Design**: Fully responsive layout built with Bootstrap 5 and custom CSS.
- **Interactive UI**: Features smooth transitions, progress tracking, and a polished glassmorphism aesthetic.


## Setup & Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/modern-resume-studio.git
   ```

2. **Open the project**:
   Simply open `index.html` in your preferred web browser. No build step, Node.js, or backend server is required.
